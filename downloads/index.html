Free and open source

Provide the academic and industrial user base a open-source software framework that can be utilized readily or with minimal modifications for routine problems relating to phase transformations during materials processing.

User friendly

MicroSim GUI allows the user to create the input files and choose the solver of his choice for the execution using a very interactive user-friendly interface. And also gives inbuild post-processing tools

Cross platform

Run on your favorite operating system as it is compatible with most of the operating system. And can be implemented on varies high-performance computing infrastructure such as MPI(CPU), CUDA(GPU) and OpenCL(GPU).

MicroSim

MicroSim is a project under the National Supercomputing Mission, Govt of India. The project offers a set of codes that can use high performance computing to simulate microstructure evolution using the Phase Field technique.
